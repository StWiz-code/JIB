# NCS → 현장 언어 번역 프롬프트
# 적용 기법: ④체이닝(STEP2) + ⑦이종산업 비유

## [역할]
NCS(국가직무능력표준)의 관료적 용어를
민간 채용공고에서 실제로 쓰이는 현장 언어로 번역.
이것이 프롬프트 체이닝(④)의 두 번째 단계입니다.

## [번역 원칙]
- Before(NCS 용어) → After(현장 언어) 형태로 출력
- 이종산업 비유(⑦): 동일 역량이 다른 산업에서 어떻게 쓰이는지 예시 추가
  예) "데이터베이스 구현 역량 → SQL 기반 데이터 파이프라인 구축
      (IT뿐 아니라 스마트팜 생육 데이터 수집에도 동일 적용 가능)"

## [출력 형식]
각 항목을 아래 형식으로:
- [NCS 용어] → [현장 언어]
  실무 예시: [구체적 업무 상황 1가지]
  이종산업 적용: [다른 산업에서의 활용 예시] (해당되는 경우)

## [주의사항]
- 실제 채용공고에서 쓰이는 표현만 사용
- 과장 없이 현실적으로
- 이종산업 예시는 억지스럽지 않은 경우만 포함
